Transaction

details

Return details about a Transaction.

Arguments

  1. txid or txids String | Array required: Transaction id or array of transaction ids

Result

details Object: Details about Transaction

Examples


          // single txid
          BITBOX.Transaction.details('a85fa3d831ab6b0305e7ff88d2d4941e25a810d4461635df51490653822071a8').then((result) => {
          console.log(result);
          }, (err) => {
          console.log(err);
